# Student Finance Tracker

Simple expense tracker for college students with AI spending predictions. Track daily expenses by category and get smart budgeting insights.

## Quick start

Requirements: Node.js (recommended via nvm) and npm.

```powershell
# Clone
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install
npm ci

# Start dev server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build for production with:

```powershell
npm run build
npm run preview
```

Replace or remove the placeholder images and metadata in `index.html` as needed for your deployment.
